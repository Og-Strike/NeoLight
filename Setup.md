## 6. Setup Instructions

Follow these steps to set up and run the smart motion-based lighting system:

1. **Upload the `.ino` file to your ESP32**
   - Use the Arduino IDE or PlatformIO to flash the ESP32 with your project code.

2. **Run the server and `data.py` script**
   - Ensure all dependencies are installed.
   - Start your backend server and execute `data.py` to handle data processing and communication.

3. **Open the Flutter app**
   - Launch the Flutter app on your mobile device or emulator for real-time control and monitoring.

4. **Power the ESP32**
   - Connect the ESP32 to a power source (USB or external power supply).

5. **Done!**
   - Your smart lighting system should now be fully functional.
